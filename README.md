## Getting Started

Clone the repository

```bash id="cloneproj"
git clone https://github.com/HetrajsinhChauhan/team-winarc-project.git
```

Install dependencies

```bash id="installproj"
npm install
```

Run development server

```bash id="runproj"
npm run dev
```
