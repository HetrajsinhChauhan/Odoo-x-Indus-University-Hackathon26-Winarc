# 🚀 Getting Started

Clone repository

```bash id="clone"
git clone https://github.com/HetrajsinhChauhan/team-winarc-project.git
```

Navigate to project

```bash id="navigate"
cd team-winarc-project
```

Install dependencies

```bash id="install"
npm install
```

Run development server

```bash id="run"
npm run dev
```
